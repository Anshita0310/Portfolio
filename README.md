# 🌐 Personal Portfolio Website

Welcome to my personal portfolio repository! This website serves as a digital resume, a gallery of my projects, and a way for people to connect with me. It showcases my skills, experience, and journey as a developer.

🖼️ **[Live Demo Link](https://anshita0310.github.io/Portfolio)**

---

## 🚀 Features

- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **About Me:** A brief introduction to who I am, my passion, and my career goals.
- **Projects Showcase:** Cards/sections detailing my best work with direct links to live demos and source code.
- **Skills Section:** A visual breakdown of the programming languages, frameworks, and tools I use.
- **Contact Form/Links:** A direct way for recruiters, clients, or collaborators to get in touch with me.

---

## 🛠️ Built With

This portfolio was crafted using the following technologies:

- **HTML5** & **CSS3** - For layout structure and custom responsive styling.
- **JavaScript (ES6+)** - For interactive elements, animations, and dynamic content.
